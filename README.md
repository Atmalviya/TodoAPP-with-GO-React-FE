## Overview
The frontend of the Todo application is built with React. It interacts with the backend API to provide a user interface for managing todo tasks.

## Technologies
- React
- Axios
- CSS
- Vercel (for deployment)

## Deployment
The frontend is deployed on Vercel.

Frontend URL: `https://todo-app-with-go-react-fe.vercel.app/`

## Setup

### Prerequisites
- Node.js and npm installed
- Backend server (running locally or deployed)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Atmalviya/TodoAPP-with-GO-React-FE.git
   cd TodoAPP-with-GO-React-FE

2. **Install dependencies:**
   ```bash
   npm install

## Run the frontend application:
   ```bash
   npm start
